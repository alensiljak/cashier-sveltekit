# OPFS Files

File Storage gives you direct access to Cashier's private, in-browser storage (OPFS) — the files backing your journal, and anything else you've saved into it.

## Browsing and editing

Tap a file to open it in the text editor.

## Adding files

From the toolbar menu:

- **New File** — creates an empty text file with a name you choose.
- **Upload File** — picks a file from your device and saves it into OPFS.

You can also drag and drop a file directly onto the page to save it. If a file with the same name already exists, you'll be asked to **Overwrite** it, **Create a Copy** (saved with a ` (1)` suffix), or cancel.

## Deleting

Use the delete icon next to a file to remove just that file, or **Delete All** from the toolbar menu to wipe every file and folder from OPFS. Both ask for confirmation first — Delete All cannot be undone.
