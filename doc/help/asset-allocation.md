# Asset Allocation

Asset Allocation shows your target allocation across asset classes alongside your current holdings, so you can see how far you've drifted from plan.

The table shows, per asset class: Target %, Current %, and Diff %, plus the equivalent Allocated, Current, and Difference values. On narrow screens, use the **%** / **###** toggle above the table to switch between the allocation view and the value view; on wider screens both are shown side by side. Tap a row with children to expand or collapse it.

## Menu options

- **Edit** — opens the Asset Allocation editor to change target percentages and asset class groupings.
- **Export** — downloads the current report as a text file.
- **Validate** — checks that the sums of each asset class group match the allocation of its parent group, and reports any mismatch.

The footer shows when the allocation data was last loaded; tap the refresh icon there to reload it.
