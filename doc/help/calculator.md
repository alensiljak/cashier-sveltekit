# Calculator

A basic arithmetic calculator, useful for working out an amount before entering it elsewhere.

The display has two lines: a small line at the top showing the expression as you build it, and a large line below showing the current number or result.

## Buttons

- **Digits 0-9** and **.** — build up a number.
- **+**, **-**, **×**, **÷** — arithmetic operators.
- **(** and **)** — group parts of the expression.
- **AC** — clears the expression and starts over.
- **⌫** — deletes the last character entered.
- **+/-** — toggles the sign of the current number (or of the result, if one is showing).
- **%** — divides the current number (or the result) by 100.
- **=** — evaluates the full expression and shows the result.

## Using the result

You can reach the calculator two ways:

- From the navigation drawer, as a standalone tool.
- From an **Amount** field in the [Journal Entry](/help/transaction-editor) editor or a balance adjustment screen, via a calculator button next to the field.

When opened from an amount field, tap the checkmark button (bottom-right) to send the current display value back to that field and return to the previous screen. When opened directly from the navigation drawer, the checkmark button has no effect.
