# Export

The Export screen turns on-device records into text you can copy, share, or download. It's a shared screen — the toolbar title and the exported content change depending on where you came from.

## Journal

Open Export from the **Export** button on the Journal card on the Home screen, or from **Export** in the Journal's toolbar menu (⋮).

Transactions are exported in Ledger format, ready to append to an existing ledger/beancount file.

## Scheduled Transactions

Open Export from **Backup** in the Scheduled Transactions toolbar menu (⋮).

Scheduled transactions are exported as JSON, for backup purposes. A file created this way can be brought back in with **Restore** on the Scheduled Transactions screen.

## The export screen

The exported content appears in a text box, which you're free to edit before sending it anywhere. Below it:

- **Copy** — copies the content to the clipboard.
- **Share** — sends it through your device's share sheet (shown only when your browser supports sharing).
- **Download** — saves it to a file named after the record type and the current date and time.

The toolbar menu (⋮) has **Sort by Date**, which reorders the entries chronologically. For a Journal export, the sorted result is also saved back to the on-device journal.
