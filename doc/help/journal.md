# Journal

The Journal lists every transaction stored on this device, in order, with the most recent at the bottom — the list scrolls to the end automatically when it loads.

Tap a transaction to open [Transaction Actions](/help/xact-actions), where you can view, edit, duplicate, schedule, or delete it.

Tap the **+** button (bottom-right) to record a new transaction.

Use the toolbar menu (⋮) for:

- **Export** — export the on-device journal.
- **Delete All** — permanently erase every transaction stored on this device (asks for confirmation first).
- **Import Ledger item** — import a transaction from a ledger file.
