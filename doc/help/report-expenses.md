# Expenses Report

The Expenses report charts all Expense-account activity for a selected time period, as either a bar chart or a donut chart. Tap a bar or slice to jump to [Transaction Search](/help/tx-search) filtered to that account and period. Use the toggle above the chart to switch between bar and donut.

## Time period

Use the **Period** dropdown to pick the timeframe the report covers.

## Filtering

The funnel icon in the toolbar toggles the expense filter on and off — when on, accounts you've hidden are excluded from the chart and total. The list icon next to it opens the filter settings page, where you check off which Expense accounts to hide. The filter turns on automatically once you hide at least one account.
