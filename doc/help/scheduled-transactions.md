# Scheduled Transactions

The Scheduled Transactions list shows every recurring or future transaction you've scheduled, sorted by next due date (and then by payee).

Each entry shows the payee, the first line of any remarks, and the amount — colour-coded red if negative, yellow if zero, green if positive. A "~" before the amount means it was auto-calculated (e.g. a repayment). The date header above each group of entries is colour-coded by how soon it's due: green in the future, yellow for today, red if overdue.

Tap an entry to open [Scheduled Transaction Actions](/help/scx-actions), where you can enter, skip, edit, or delete it.

Tap the **+** button (bottom-right) to add a new scheduled transaction.

## Search

The search bar below the toolbar filters the list by payee and remarks as you type.

## Toolbar menu

Use the toolbar menu (⋮) for:

- **Backup** — export your scheduled transactions to a JSON file (copy, share, or download).
- **Restore** — import scheduled transactions from a previously exported JSON file (asks for confirmation first).
- **Calendar** — open the [Calendar](/help/scheduled-calendar) view of scheduled transactions.
