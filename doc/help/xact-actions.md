# Transaction Actions

This screen shows a transaction's details along with the actions available for it.

## Actions

- **Duplicate** — copies this transaction into a new, editable transaction (same postings and details, dated today) and opens it in the transaction editor.
- **Schedule** — creates a new scheduled transaction from this one and opens the scheduled transaction editor, so you can set up a recurring version of it.
- **Copy** — copies the transaction as Beancount-formatted text to the clipboard.

Two more actions are available only for transactions stored on this device (not for transactions read from your full Beancount ledger):

- **Edit** — opens the transaction in the transaction editor to change it.
- **Delete** — removes the transaction from the device journal, after asking for confirmation.

You can reach this screen from the [Journal](/help/journal) or from [Account Transactions](/help/account-transactions).
